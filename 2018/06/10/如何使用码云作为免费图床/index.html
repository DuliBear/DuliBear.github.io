如何使用码云作为免费图床

什么是图床？图床就是用来存放图片的空间，让你上传图片并能够分享到互联网。网上有很多优秀的图床，比如七牛图床，但是需要实名认证和付费使用，虽然稳定但是未免有些麻烦。后来我又了解到了新浪微博图床，完全免费而且上传图片极为简单。然而按照新浪的尿性，说不定哪天就出现“由于政策原因”的问题，微博图床就凉了，到那时就真的欲哭无泪了…也不推荐GitHub作为图床，国内访问速度慢，而且万一以后被墙了呢？

所以这里我就介绍一下如何将中国版GitHub——码云作为免费的图床了（免费对于我来说真的是很吸引了）

第一步

假设你已经学会如何在码云上新建仓库，并且了解相关的 git 知识。如果不，我推荐你看 廖雪峰老师的 Git 教程，另外再附上我看过从一本很好的 0 开始学习 GitHub 电子书。感谢廖雪峰老师和此书的作者，让我接触到伟大的 GitHub！如果看不懂以下教程话，请务必先熟悉 git 命令！话不多说，入正题吧！

  • 首先创建一个项目，我将其命名为 giteePic
  • 将你的本地图片推送到仓库里面
#分别执行以下语句上传图片，以上传 img_1.png 为例
$ git add img_1.png //也可以执行 "git add ." 添加所有文件
$ git commit -m "push img_1"
$ git push -u gitee master

第二步

怎么使用你已经 push 到仓库的图片？举个例子：

    https://gitee.com/DuliBear/giteePic/blob/master/img_1.png

这个是你图片的远程地址，如果你访问这个地址的话只是访问到相应的仓库地址，这时我们并不能在其它网站使用这张图片，如下图所示

接着，你只需要将刚刚链接中的 blob 换成 raw ，再访问一下，看看是不是效果不同了？这时你就能在互联网使用你的图片了嘻嘻 (●’◡’●)
